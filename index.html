Retro Pedals (TBD)

A passion project of mine to recreate old guitar pedal sounds. Modern guitar pedal circuits somehow sound noticably worst than old effects. This is due to evolved construction of circuit components such as capacitors and resistors, but a huge part is also how these circuits are wired.

A software driven digital effects pedal which utilizes Python and a Raspberry Pi to create an audio interface suite, right by your foot. Since there is only so much you can do on a tiny touchscreen, the pedal software will also have a desktop GUI accessible via bluetooth.

Deadbeat Simulator

Game development class project, where my team and I were task to create a game of our liking in Godot. Deadbeat Simulator is an overtly ambitious project which takes after the likes of the original Grand Theft Auto video games. This was a showcase of our skills and learning in Godot.

Senior Project: Theremin w/ Digital Effects

Worked in tandem with an Electrical Engineer to produce a Theremin. The main goal was to design a theremin which would have modular effects using an ADC+DAC audio interface attached to a Raspberry Pi. Digital signal processing was done using equations to emulate different signal filters (such as LPF, HPF, etc) used in conjunction with Python libraries to perform this DSP.
